base Module¶

class ebu_tt_live.node.base.Node(node_id, carriage_impl)[source]¶

Bases: object

This is the foundation of all nodes that take part in the processing of subtitle documents. The Node should deal with subtitles in a high level interface, which is an instance of <ebu_tt_live.documents.SubtitleDocument>. That is the interface which should be used to communicate with the carriage machanism. See <CarriageImpl>
